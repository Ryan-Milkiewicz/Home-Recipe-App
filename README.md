# 🍳 Home Recipe App

A simple, clean, and customizable app for managing your personal recipe collection. Whether you're organizing family favorites or experimenting with new dishes, this app helps you keep everything in one place.

---

## ✨ Features

* 📖 **Create & Store Recipes**

  Add ingredients, instructions, cooking time, and notes.
* 🚀 **Scrape Recipes**

  Auto-populate all fields from the web.
* 🔍 **Search & Filter**

  Quickly find recipes by name.
* 🏷️ **Categories & Tags**

  Organize meals by cuisine, dietary preference, or occasion.
* ⭐ **Favorite Recipes**

  Save your go-to meals for quick access anytime.
* 📱 **Responsive Design**

  Works on desktop, tablet, and mobile devices.
---

## 🚀 Getting Started

### Prerequisites

Make sure you have the following installed:

* Node.js
* npm or pnpm

### Installation

```bash
# Clone the repository
git clone https://github.com/your-username/home-recipe-app.git

# Navigate into the project folder
cd home-recipe-app

# Install dependencies
npm install

# Start the development server
pnpm start
```

The app should now be running at `http://localhost:3000`.
